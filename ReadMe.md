# Vacancy Parser
Это приложение Spring Boot, предназначенное для анализа вакансий с различных популярных российских сайтов по поиску работы. Оно агрегирует данные и предоставляет простой API для доступа к результатам.
## Особенности
- **Анализ нескольких сайтов:** поддерживает анализ вакансий со следующих сайтов:
  - [Habr Career](https://career.habr.com)
  - [HH.ru](https://hh.ru)
- **Полная пагинация результатов:** парсер извлекает все вакансии со всех страниц результатов поиска, а не только с первой страницы.
- **Асинхронный парсинг:** задачи парсинга выполняются в фоновом режиме, чтобы не блокировать запросы API.
## Используемые технологии
- **Java 17**
- **Spring Boot 3.5.9**
- **Maven**
- **Spring Data JPA** (с базой данных H2 в памяти)
- **Lombok**
- **Jsoup** (для скрапинга HTML)
- **Jackson** (для анализа JSON)
- **Spring Web (RestTemplate)** (для отправки HTTP-запросов)

## Как использовать:
1.  **Запуск приложения:**

В IDE запустите VacParserApplication.java, по пути: VacParser\src\main\java\com\example\VacParser\VacParserApplication.java
Приложение запустится по адресу `http://localhost:8080`.

2.  **Использование API:**

В файле request.http укажите ссылку на нужный сайт (примеры уже есть в файле) и отправьте запрс.
Когда все вакансии будут обработаны, открываем в браузере: http://localhost:8080/h2-console/login.jsp и войдя в бд (application.properties) получаем доступ к вакансиям.
